# Jeeva Raksha — Unified Hospital Information System

ಜೀವರಕ್ಷ (Jeeva Raksha) — ಜೀವ ಉಳಿಸುವ ತಂತ್ರಜ್ಞಾನ. A trust-driven, AI-powered digital backbone for hospitals, combining clinical intelligence with healing-centric design.

## Features

- **OPD & IPD Management** — Streamlined outpatient and inpatient workflows
- **Emergency & OT Management** — Real-time emergency and operation theatre control
- **Doctor's Pad** — Digital ward rounds with stylus support and AI-assisted notes
- **Laboratory & Radiology** — AI-powered lab report analysis and radiology scan interpretation
- **Bed Management** — Live bed occupancy tracking and allocation
- **Billing & Inventory** — Comprehensive billing, pharmacy, and inventory management
- **Patient Portal** — Patient-facing portal for records, appointments, and reports
- **Multi-language Support** — English, Kannada, and Hindi

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```
   npm install
   ```
2. Set the `API_KEY` in `.env.local` to your AI API key
3. Run the app:
   ```
   npm run dev
   ```
4. Start the backend server:
   ```
   npm run server
   ```

## Tech Stack

- **Frontend:** React, TypeScript, Vite, Recharts
- **Backend:** Express.js, PostgreSQL
- **AI:** Clinical AI for diagnostics, lab analysis, and radiology interpretation
